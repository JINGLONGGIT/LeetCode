### 题目分析
* 边界条件判断：数组大小为0或者为1时，直接返回即可
```C++
    if (nums.size() == 0 || nums.size() == 1)
    {
        return nums.size();
    }
```

***
#### 解法一
* 思路：给定两个索引，从第二个元素开始，将当前元素依次和前一个元素进行比较
```C++
    i = j = 1;
    if (nums[i - 1] != nums[i])
    {
        nums[j++] = nums[i];
    }
```
* 时间复杂度：O(N)
* 空间复杂度：O(1)
