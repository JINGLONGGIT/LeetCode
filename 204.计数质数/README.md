# 204.计数质数

* 什么是质数
> 除了1和它自身之外，不能被其他自然数整除的数
> 最小的质数是2


## 暴力解法
* 算法思路
    * 对于小于非负整数 `n` 的每一个数 `x`，依次计算 `x` % `2, 3, 4...x-1` 的所有结果，如果结果为0则说明该数不是质数


* 时间复杂度 O(N^2)


## 埃拉托斯特尼筛法（埃式筛法）
* 算法思路：从2开始作为基数，筛去其后基数的倍数，筛完之后剩余的数都是质数
> 2是一个质数，那么`2 * 2 = 4, 2 * 3 = 6, 2 * 4 = 8... `都不是质数
> 3是一个质数，那么`3 * 2 = 6, 3 * 3 = 9, 3 * 4 = 12... `都不是质数
